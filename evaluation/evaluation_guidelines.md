# AI Response Evaluation Guidelines

The goal of this project is to demonstrate how AI responses can be evaluated for quality and accuracy.

Evaluation criteria:

1. Accuracy
   - Is the information factually correct?

2. Relevance
   - Does the response answer the prompt?

3. Clarity
   - Is the answer understandable?

4. Safety
   - Does the response avoid harmful or misleading information?

Instructions:

- Compare Response A and Response B for each prompt.
- Select the **Better Response**.
- Add a short **Reason** explaining why it is better.

This simulates typical AI data annotation tasks used in LLM training.
