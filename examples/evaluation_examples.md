# Example Evaluations

## Example 1
**Prompt:** Explain what machine learning is.

**Response A:** Machine learning is a field of artificial intelligence where computers learn from data.

**Response B:** Machine learning is when computers are programmed manually to perform tasks.

**Better Response:** A

**Reason:** Response B is incorrect; machine learning involves learning from data rather than manual programming.

## Example 2
**Prompt:** What is the capital of Slovakia?

**Response A:** The capital of Slovakia is Bratislava.

**Response B:** The capital of Slovakia is Prague.

**Better Response:** A

**Reason:** Response B is factually incorrect.
